# Google実装パッケージ

PTA適正運営スターターキット v1.0 のGoogleフォーム・スプレッドシート・Apps Script実装方針です。

## 作るもの

- 入会申込フォーム
- 退会通知フォーム
- 会員情報更新フォーム
- 役員立候補・承諾フォーム
- 会員台帳
- 会費管理
- 退会管理
- 役員承諾
- 権限管理
- 削除予定
- ログ

## 作らないもの

- 非加入届フォーム
- 加入・非加入二択フォーム
- 未提出者督促
- 学校名簿インポート
- 学校名簿照合
- 非会員児童一覧
- 役員免除申請
- 家庭事情申告

## 原則

会員台帳は、入会申込に基づく加入者のみで作成します。学校が保有する名簿をPTA会員管理に使いません。
