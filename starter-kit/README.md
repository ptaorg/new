# PTA適正運営スターターキット v1.0

発行日：2026年7月3日  
発行：PTA適正化推進委員会  
公開入口：`/starter-kit/`

## 位置付け

PTAの任意加入、個人情報保護、会計分離、学校との公私分離を整理したスターターキットです。

## 公開ページ

- `index.html`：公開入口
- `manual.html`：マニュアル本文
- `templates.html`：標準書式14種
- `rules.html`：PTA会則モデル
- `roadmap.html`：移行ロードマップ
- `school-board.html`：学校・教育委員会向け説明書
- `rebuttals.html`：反論対応集
- `audit.html`：情報公開・監査確認項目
- `sources.html`：一次資料リンク集
- `qa.html`：QA・検査結果
- `changelog.html`：更新履歴
- `style.css`：共通スタイル

## 原則

入会する人だけが申込書を出します。入会しない人に非加入届や辞退届を求めません。PTA会員台帳は、PTAが本人から取得した情報で作成します。学校名簿、学校徴収金、学校連絡網、担任回収、児童経由回収に依存しない運営へ移行します。

## 今後の扱い

PDF本体、ZIP本体、Apps Script本体、スプレッドシート雛形などの大容量・バイナリ系ファイルは、GitHub Pages直置きではなく、必要に応じてRelease添付または別ディレクトリで管理します。
