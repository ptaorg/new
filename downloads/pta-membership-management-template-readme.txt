PTA加入・会員管理テンプレート
================================

この配布セットは、加入を希望する本人から得た情報を基に、PTAが会員管理を行うためのひな形です。
学校名簿をPTA会員名簿として流用したり、申込のない方を会員として登録したりする用途には使用しないでください。

収録内容
--------
- pta-membership-management-template.csv
  会員管理用の空のCSVひな形です。申込受付、重要事項同意、会費確認、会員確定を分けて記録できます。
- pta-membership-template.html
  加入手続と会員管理の考え方、利用の流れを説明するページです。
- pta-membership-template-admin.html
  PTA役員・管理者向けの運用上の注意をまとめたページです。
- pta-membership-inquiry.pdf
  入会手続について確認する際に使えるPDFです。
- pta-school-separation-notice-template.pdf
  学校とPTAの事務を分けるための通知文テンプレートです。

利用上の注意
------------
1. CSVは必要最小限の項目だけを残し、各PTAの会則・会費規程に合わせて調整してください。
2. 会員情報へアクセスできる担当者を限定し、保存場所の共有権限を定期的に確認してください。
3. 会員確定前の申込情報と、確定後の会員名簿を区別してください。
4. 保管期間を定め、不要になった個人情報は安全に削除してください。
5. HTML内の外部リンクを利用する場合は、インターネット接続が必要です。
