# トップページ構成棚卸し

## 現在の主要構成

1. 支援告知帯
2. ヘッダー・検索・ナビゲーション
3. スマホ用メニュー
4. 写真スライド型ヒーロー
5. Main Issue セクション
6. Evidence to Action セクション
7. 現場レポート動画セクション
8. 一次資料・行政見解セクション
9. 適正運営ガイドセクション
10. このサイトの歩き方
11. 最新更新・掲載情報
12. フッター

## 良い点

- ヒーロー直下に「問題はPTAがあることではなく、学校手続に溶け込むこと」という中核メッセージがある。
- 保護者、学校・教育委員会、資料を見る、という初動導線がヒーローにある。
- 主要論点として、入会、会費、情報・人・場所の3分類がある。
- 回答データベース、全国資料館、適正運営ガイドへの導線がある。
- `site.js` は `v=89` に更新済みで、本文はHTML側が正本になっている。

## 改善が必要な点

### 1. PTA役員向け導線がヒーローにない

現在のヒーロー導線は、保護者、学校・教育委員会、資料の3つ。
PTA役員向けの入口はナビにはあるが、初見では見落としやすい。

改善案：

- ヒーロー導線に `PTA役員として読む` を追加する。
- ただしボタンを増やしすぎるとスマホで窮屈になるため、3入口カードをヒーロー直下に設ける方法も検討する。

### 2. 「資料を見る」が強すぎる

初見ユーザーは、資料より先に自分の立場を選ぶ方が迷わない。
資料導線は重要だが、第一導線ではなく第二導線でもよい。

改善案：

- 第一導線：保護者／PTA役員／学校・教育委員会
- 第二導線：教育委員会回答／全国資料館／資料入口

### 3. 最新更新の内容が古く見える可能性

令和7年・令和8年の更新が混在しており、初見では現在更新されているサイトか判断しづらい。

改善案：

- 「最新更新」には最終更新日を明示する。
- 古いが重要な項目は「主要資料」側へ移す。

### 4. セクションの順番がやや資料寄り

現在は、問題提起の後にすぐ Evidence to Action と資料導線が来る。
保護者・役員・学校の立場別入口をもう少し上に置いた方が分かりやすい。

改善案：

1. ヒーロー
2. 立場別3入口
3. Main Issue
4. 主要論点
5. 資料・回答DB
6. 適正運営ガイド
7. 最新更新
8. 支援導線

### 5. トップページ内CSSが大きい

`index.html` 内に大量のCSSがある。すぐに外部化すると事故リスクがあるため、本文改善後に段階的に整理する。

## 次に触るなら

大型HTMLのため、次の順で小さく進める。

1. ヒーロー直下に立場別3入口が既にあるか確認する。
2. ない場合、短い独立セクションとして追加する。
3. 追加は本文ブロック単位で行い、既存CSSを極力再利用する。
4. 追加後に `npm run generate:search` と `npm run generate:sitemap` を実行する。

## 注意

- `index.html` の全文置換は避ける。
- トップページ本文を `site.js` で差し替えない。
- 画像や動画の重い追加はしない。
- 最初の改善は、文章と導線だけに絞る。
