# 01 全体像

このセットは、PTAのホームページ、入会申込フォーム、役員用管理アプリを組み合わせて運用するためのテンプレートです。

## 保護者が見るもの

- Googleサイト
- 入会申込フォーム

保護者には、管理スプレッドシートや管理アプリURLを共有しません。

## 役員が使うもの

- PTA管理アプリ付きスプレッドシート
- Apps Script 管理アプリ画面

役員は、スプレッドシートのセルを直接編集するのではなく、管理アプリ画面から申込確認・会員確定・会員名簿確認を行います。

## データの流れ

```text
Googleサイト
↓
入会申込フォーム
↓
フォーム回答シート
↓
申込管理
↓
会員確定
↓
会員名簿
```
