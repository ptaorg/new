# 03 役員用管理アプリの入れ方

## 使うファイル

- `apps-script/Code.combined.gs`
- `apps-script/Index.html`

## 手順

1. 管理用スプレッドシートを開く。
2. `拡張機能` → `Apps Script` を開く。
3. `Code.gs` の中身を全部消す。
4. `apps-script/Code.combined.gs` の中身を全部貼る。
5. 左側の `＋` から `HTML` を追加する。
6. ファイル名を `Index` にする。
7. 作成された `Index.html` に `apps-script/Index.html` の中身を全部貼る。
8. 保存する。
9. スプレッドシートを再読み込みする。
10. メニュー `PTA管理` → `管理アプリ画面を開く` を押す。

## Webアプリとして公開する場合

まずは自分だけに公開してください。

```text
デプロイ
↓
新しいデプロイ
↓
種類：ウェブアプリ
↓
実行するユーザー：自分
↓
アクセスできるユーザー：自分のみ
↓
デプロイ
```

## 共有上の注意

- 保護者に管理アプリURLを共有しない。
- 保護者に管理スプレッドシートURLを共有しない。
- 編集用フォームURLを保護者に共有しない。
