# 04 やってはいけないこと

このテンプレートは、任意加入・明示申込・学校名簿非流用を前提にしています。

## 禁止すべき運用

- 加入しない方にフォーム提出を求めること。
- 非加入届として使うこと。
- 学校名簿をPTA会員名簿として流用すること。
- 申込のない保護者を会員として登録すること。
- 加入しない児童・生徒に不利益を与えること。
- 管理アプリURLや管理シートURLを保護者全体に共有すること。
- 編集用フォームURLを公開すること。

## サイトやフォームに必ず入れる文言

```text
PTAへの加入は任意です。
加入しない方は、入会申込フォームを提出する必要はありません。
加入しないことにより、児童・生徒が学校教育活動上の不利益を受けることはありません。
このフォームは、PTAへの加入を希望する方だけが提出するものです。
```
