# 02 GoogleサイトHTMLの使い方

Googleサイトでページを作った後、各ページにHTMLを埋め込みます。

## 対応表

| Googleサイトのページ | 貼るHTML |
|---|---|
| トップページ | `google-sites-html/01_top.html` |
| お知らせ | `google-sites-html/02_news.html` |
| 活動紹介 | `google-sites-html/03_activities.html` |
| 資料室 | `google-sites-html/04_documents.html` |
| 入会案内 | `google-sites-html/05_join.html` |
| 重要事項 | `google-sites-html/06_important.html` |
| FAQ | `google-sites-html/07_faq.html` |
| お問い合わせ | `google-sites-html/08_contact.html` |

## Googleサイトでの操作

```text
挿入
↓
埋め込む
↓
埋め込みコード
↓
HTML全文を貼り付ける
↓
挿入
```

## 差し替えるもの

- PTA名
- 学校名
- 年度
- 会費
- 支払い方法
- 問い合わせ先メール
- 入会申込フォームURL
- PDF資料リンク
